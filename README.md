# Personal Blog

The blog uses [Hugo](https://gohugo.io/). For installation instructions follow [this guide](https://gohugo.io/getting-started/quick-start/).

## Setup

```
$ git clone git@github.com:mgechev/blog.mgechev.com
$ cd blog.mgechev.com
$ hugo server
```

Open http://localhost:1313/.
